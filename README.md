# Flipping Card UI Design

A sleek, glassmorphism-inspired credit card UI design featuring a smooth 3D flipping animation on hover. Built with HTML and CSS, this project demonstrates advanced CSS techniques including `perspective`, `backface-visibility`, and `backdrop-filter`.

## Features

- **3D Flip Animation:** Interactive card that flips to reveal the back side on hover.
- **Glassmorphism Effect:** Modern frosted glass look using `backdrop-filter: blur()`.
- **Responsive Layout:** Centered card design using Flexbox.
- **Detailed Design:** Includes credit card essentials like chip, logo, magnetic strip, and signature area.
- **Custom Typography:** Integrated with Google Fonts (Poppins).

## Project Structure

```text
Flipping Card UI Design/
├── index.html      # Main HTML structure
├── style.css       # Custom CSS styling and animations
└── images/         # Project assets

    └── logo.png    # Master Card / Branding logo
```

## How to Run

Simply open `index.html` in any modern web browser to view the design.
